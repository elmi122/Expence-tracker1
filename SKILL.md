# Google Gallery Edge Expense Tracker

Build a polished interactive expense tracker with card-based layout, bold category colors, smooth animations, and a dashboard overview.

## Categories
- Food: 🍔 #f97316
- Transport: 🚗 #3b82f6
- Housing: 🏠 #8b5cf6
- Health: 💊 #22c55e
- Entertainment: 🎮 #ec4899
- Shopping: 🛍️ #f59e0b
- Work: 💼 #6366f1
- Travel: ✈️ #06b6d4
- Other: 📦 #94a3b8

## Features
- Dashboard header with month navigation and totals
- Expense card gallery with category filter tabs
- Quick add form with income/expense toggle
- SVG donut chart analytics panel
- Floating + button for quick add
- Pre-loaded sample data on first load

## Implementation
Single React JSX artifact, all state in useState, no localStorage, no external chart libraries.
